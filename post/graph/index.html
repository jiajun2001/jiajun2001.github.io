Graph Algorithms

  • 1/10/2022
  • 2-minute read

Graph algorithms are used to solve the problems of representing graphs as networks like airline flights, how the Internet is connected, or social network connectivity on Facebook. They are also popular in NLP and machine learning fields to form networks (Ejonavi 2020). In this post, I will talk about some important graph algorithms with basic ideas under the hood, specific implementations, and complexity analysis.

Breadth First Search (BFS)

Idea for breadth-first-search: Start from node s and visit all nodes with distance i to s in iteration i.

Implementation:

Class Declaration

class Graph {
 private:
    int nodeNum;
    vector<list<int>> adjacencyList;
 public:
    // Constructor
    Graph(int nodeNum); 

    // Add edges to the graph
    void addEdge(int parent, int child);

    // Breadth First Search
    void BFS(int startingNode);
};

Method Implementation

Graph::Graph(int nodeNum) {
    this->nodeNum = nodeNum;
    adjacencyList.resize(nodeNum);
}

void Graph::addEdge(int parent, int child) {
    // There is an edge from 'parent' to 'child'
    adjacencyList[parent].push_back(child);
}

void Graph::BFS(int startingNode) {
    // Initialization
    vector<vector<int>> outcome;
    vector<bool> visited(nodeNum, false);
    queue<int> myQue;

    // Starting with 'startingNode'
    myQue.push(startingNode);
    visited[startingNode] = true;

    while (!myQue.empty()) {
        vector<int> oneLayer;
        int size = myQue.size();
        for (int i = 0; i < size; i++) {
            // Handle current node
            int s = myQue.front();
            oneLayer.push_back(s);
            myQue.pop();
            // Handle neighboring nodes
            for (auto node : adjacencyList[s]) {
                if (!visited[node]) {
                    visited[node] = true;
                    myQue.push(node);
                }
            }
        }
        outcome.push_back(oneLayer);
    }

    // Print the final outcome
    for (int i = 0; i < outcome.size(); i++) {
        cout << "Layer " << i << ": ";
        for (int j = 0; j < outcome[i].size(); j++) {
            cout << outcome[i][j] << " ";
        }
        cout << endl;
    }
}

Depth First Search (DFS)

Idea for depth-first-search: Whenever you visit a vertex, explore in the next step one of its non-visited neighbors.

Implementation:

Class Declaration

class Graph {
 private:
    vector<bool> visited;
    vector<list<int>> adjacencyList;
 public:
    // Constructor
    Graph(int nodeNum);
    // Add edges to the graph
    void addEdge(int parent, int child);
    // Depth First Search
    void DFS(int startingNode);
};

Method Implementation

Graph::Graph(int nodeNum) {
    visited = vector<bool>(nodeNum, false);
    adjacencyList.resize(nodeNum);
}

void Graph::addEdge(int parent, int child) {
    // There is an edge from 'parent' to 'child'
    adjacencyList[parent].push_back(child);
}

void Graph::DFS(int startingNode) {
    // Handle current node
    cout << startingNode << " ";
    visited[startingNode] = true;
    // Handling neighboring nodes
    for (auto n : adjacencyList[startingNode]) {
        if (!visited[n]) {
            DFS(n);
        }
    }
}
